# Fotos del deck

Deja los archivos con EXACTAMENTE estos nombres en esta carpeta y aparecerán
solos en las slides (no hay que tocar código). Hasta que los pongas, se ve un
recuadro con el texto del hueco.

| Archivo                     | Slide                          | Sugerencia de foto                          |
|-----------------------------|--------------------------------|---------------------------------------------|
| `harvard.png`               | Portada                        | Foto del equipo en Harvard / 2º mundial     |
| `product.png`               | El producto                    | Captura del dashboard o del juego en acción |
| `gregorio-maranon.jpeg`     | Fase 3 (primer piloto)         | Fachada o interior del Gregorio Marañón     |
| `la-paz.jpeg`               | Fase 4 (segundo sitio)         | Hospital La Paz                             |

Formatos válidos: `.jpg`, `.png`, `.webp` (si cambias la extensión, avísame para
ajustar la ruta). Ideal horizontal, mínimo ~1200px de ancho.
